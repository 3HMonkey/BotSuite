public void IniParser(String IniPath)

Parameters

IniPath

String

file to open, either a relative or a absolute path (relative pathes start with "\" and for can only be pathes the same level as executable or below in file tree) --- UNCs not supported at the moment!

Example

// load "config.ini" from the application directory IniParser Config = new IniParser("\config.ini"); // write something Config.Write("SomeVariable", "ValueToWrite"); // read something string Get = Config.Read("SomeVariable");
