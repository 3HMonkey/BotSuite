Move(Point targetPosition, Boolean human, Int32 steps)

moves the mouse to a specific point "target" through "steps" number of points

public static Boolean Move(Point targetPosition, Boolean human, Int32 steps)

Parameters

targetPosition

Point

The target position

human

Boolean

Simulate human-like jumps

steps

Int32

The points of pathpolygons

Remarks

The more points you`ll use the more smoother the curve will be If the option "human" is false the cursor will jump to the target point and ignores "steps"

Example

Point target = new Point(10,10);
Mouse.Move(target,true,10);
