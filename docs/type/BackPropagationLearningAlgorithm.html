Summary

stockastic gradient backpropagation learning algorithm

Remarks

Neuron j in L-1
o ---------> Signal f(WS_j) Neuron i in L o ---------> Signal f(WS_i) Neuron k in L+1

for i

ti(n+1) = ti(n) - \alphaEO_i\gamma * ( Ti(n) - Ti(n-1)) w{i,j}(n+1) = w{i,j}(n) + \alphaS_jEOi+\gamma * (w[i,j}(n)-w_{i,j}(n-1))

    with :
            EO_i = f'(WSi) * (\IE Ouput of i - S_i)        for output layer
            EO_i = f'(WSi) * \sum_k  w_{k,i} EO_k  )       for others
