IdentifyColor(ImageData img, Dictionary statReference, Int32 left, Int32 top, Int32 width, Int32 height)

Identifies the colors of the pixels in a rectangle from an image by a given list of reference colors (root means square error from average)

public static Color IdentifyColor(ImageData img, Dictionary statReference, Int32 left, Int32 top, Int32 width, Int32 height)

Parameters

img

ImageData

The image that should be analyzed

statReference

Dictionary

The list of possible colors

left

Int32

The left of the rectangle (default: 0)

top

Int32

The top of the rectangle (default: 0)

width

Int32

The width of the rectangle (default: full width)

height

Int32

The height of the rectangle (default: full height)

Returns

The best matching color

Remarks

if the color differs sometimes you can use this function to find the best matching colors in a dictionary list. In most you do not need to know the exact rgb values. You only wants to know whether the rectangle is more likely to the color red, blue, rgb(?,?,?), ....

this functions maps the result to the correct color. If the image contains a pale red and you want to identify this as #ff0000 you have to add the pair (#FF8C8C "pale red", #FF0000)

This uses the RMSE root mean square error to find the best color. Attention! If there are some pixel that have to complete other colors, they can destroy the accurately see: IdentifyColorByVoting

Example

ImageData Screenshot = new ImageData(...); Dictionary> Choose = new Dictionary> (); Choose.Add(Color.Red, new List { 255, 144.23, 140.89 }); Choose.Add(Color.White, new List { 218, 219, 222 }); Choose.Add(Color.Blue, new List { 21, 108, 182 }); Choose.Add(Color.Green, new List { 86, 191, 50 }); Choose.Add(Color.Yellow, new List { 233, 203, 118 }); Choose.Add(Color.Orange, new List { 246, 122, 11 }); Choose.Add(Color.Black, new List { 94, 98, 98 }); Choose.Add(Color.Violet, new List { 223, 80, 195 }); Choose.Add(Color.MediumSeaGreen, new List { 106, 227, 216 }); // ... Color PieceColor = CommonFunctions.IdentifyColor(Screenshot,Choose,leftoffset,topoffset,width,height);

IdentifyColorByVoting(ImageData img, Dictionary statReference, Int32 left, Int32 top, Int32 width, Int32 height)

Identifies the colors of the pixels in a rectangle and compares them with those from an image by a given list of reference colors (majority vote)

public static Color IdentifyColorByVoting(ImageData img, Dictionary statReference, Int32 left, Int32 top, Int32 width, Int32 height)

Parameters

img

ImageData

The image to look in

statReference

Dictionary

The list of possible colors

left

Int32

The left of the rectangle (default: 0)

top

Int32

The top of the rectangle (default: 0)

width

Int32

The width of the rectangle (default: full width)

height

Int32

The height of the rectangle (default: full height)

Returns

The best matching color

Remarks

This tests every pixel in the given rectangle and majority votes the color from the given dictionary of possible colors. Each pixel votes for a color.

if the image patch is

" red red " " red blue "

then 4 pixels vote for similary colors to red and one pixel votes for a similar color to blue

Example

ImageData Screenshot = new ImageData(...); Dictionary> Choose = new Dictionary> (); Choose.Add(Color.Red, new List { 255, 144.23, 140.89 }); Choose.Add(Color.White, new List { 218, 219, 222 }); Choose.Add(Color.Blue, new List { 21, 108, 182 }); Choose.Add(Color.Green, new List { 86, 191, 50 }); Choose.Add(Color.Yellow, new List { 233, 203, 118 }); Choose.Add(Color.Orange, new List { 246, 122, 11 }); Choose.Add(Color.Black, new List { 94, 98, 98 }); Choose.Add(Color.Violet, new List { 223, 80, 195 }); Choose.Add(Color.MediumSeaGreen, new List { 106, 227, 216 }); // ... Color PieceColor = CommonFunctions.IdentifyColorByVoting(Screenshot,Choose,leftoffset,topoffset,width,height);
